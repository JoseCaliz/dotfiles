# dotfiles

My dotfiles for:
   1. Nvim
   2. Tmux
   3. Iterm2
   4. Zsh


Location
```
~/Library/Preferences/com.googlecode.iterm2.plist
```

## COC

I started to use coc for some time ago. Make sure to install `coc-pyright`


```
CocInstall coc-pyright
```

## Keyboard Layout

Copy the bundle to:
```
~/Library/Keyboard Layouts/
```
